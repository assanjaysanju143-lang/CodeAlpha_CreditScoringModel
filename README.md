# Credit Scoring Model - CodeAlpha Internship

## 📌 Objective
To predict whether a person is creditworthy using machine learning.

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

## ⚙️ Model Used
Logistic Regression

## 📊 Features Used
- Age
- Job
- Credit Amount
- Duration
- Housing
- Saving Accounts
- Checking Account

## 🔍 Process
- Data Cleaning
- Label Encoding
- Feature Scaling
- Model Training
- Model Evaluation

## 📈 Evaluation Metrics
- Precision
- Recall
- F1-Score
- ROC-AUC Score

## 🎯 Conclusion
The model successfully predicts credit risk based on financial attributes.

## 💻 Streamlit App
This project also includes a Streamlit web app for real-time credit risk prediction.
Run using:
streamlit run app.py
